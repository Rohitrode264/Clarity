# Clarity - YouTube Video Summarizer

Clarity is a Chrome extension that generates concise summaries of YouTube videos using AI. It extracts transcripts and provides an easy-to-read summary.

## Features
- Extracts YouTube video transcripts
- Uses AI to generate short and simple summaries
- Clean and minimal UI with smooth animations
- Built with React, TypeScript, and Tailwind CSS

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/Clarity.git
   cd Clarity
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run the development server:
   ```sh
   npm run dev
   ```
4. Load the extension in Chrome:
   - Go to `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked" and select the `dist` folder
